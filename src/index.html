Projects


Below is a list of personal projects that I have completed along with the tools and technologies used to achieve the results. Many of these projects use a JavaScript framework (R3F, THREEjs, or React) and are deployed using Vercel and either Vite or Webpack.

Since you are here I thought my Portfolio site would be a good place to start.
This Portfolio is built using THREE.js, HTML, JS, GSAP, and CSS.

The hero image utilizes the vertex and fragment glsl shaders to portray the movement of the colours and to create a reaction to any noise picked up via the microphone. The hero also utilizes the orbit camera to allow the user to rotate around the centre sphere.

The flag uses a similar shader technique although this uses time rather than sound to move the vertex and create a wave like motion. The following two 3d models are animated using GSAP based on the position of the user on the page. The stars are also randomly generated points so will change when the page is refreshed giving each user a different experience

Click here to view a more customizable experience

This game was created using react-three-fibre and using a range of different techniques. The game consists of one level that changes on each render by randomly selecting 10 play sections from an array. The array consists of three individual sections that have obstacles with randomly generated motions and start points. The physics are created by using rapier's physics engine and all geometries are created by manipulating the default THREEjs geometries (aside from the hamburger). The hamburger was created in blender. The game uses React hooks such as useEffect, useState, and useMemo.

Link to the project: Ball Run Game

This project is build using THREEjs and uses the fragment and vertex glsl shaders to create patterns that form waves. The waves change colour based on their height. I have allowed most variables to be adjusted by the user on a user friendly gui. The wave movement is based on time and give a nice illusion of a real ocean. The project also has orbit controls to allow for a completely immersive view of the waves.

Link to the project: Animated Ocean

This is another THREEjs project that utilizes the fragment and vertex shaders alongside the time function. This portal scene was created in blender and then imported using react-three-fibre to be displayed on the web. The portal itself uses a custom shader material to give it an animated look.
